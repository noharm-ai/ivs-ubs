Análise das Diferenças

IVS médio: SMS-POA 0.589 vs Demo 0.503 (Δ = -0.087). O demo subestima a vulnerabilidade média em ~0.087 pontos. Esperado: dados aleatórios tendem a centralizar ao redor da média da escala (0,5), enquanto os dados reais capturam a assimetria positiva real de Porto Alegre.
Dispersão (std): SMS 0.152 vs Demo 0.089 (Δ = -0.063). O demo tem menor dispersão, o que é consistente com dados sintéticos que não reproduzem a heterogeneidade real entre territórios vulneráveis e privilegiados de POA.
Faixa Alta (≥ 0,75): SMS 12.1% das UBS vs Demo 0.0%. Nenhuma UBS demo atinge a faixa alta, porque o gerador aleatório não concentra D1/D4 altos simultaneamente com a mesma frequência observada em territórios reais de alta vulnerabilidade (p. ex. ilhas, periferias com alta D4 = mães adolescentes).
Coerência estrutural: apesar das diferenças numéricas, o pipeline reproduz corretamente a lógica da fórmula: D1 (peso 0,50) domina o IVS, o coeficiente de correlação D1×IVS no demo (esperado ~0,85) replica o padrão do SMS 2019. O pipeline está metodologicamente válido para ser alimentado com dados reais.

O que muda com dados reais?

  • D1 e D4 mais elevados em territórios de periferia → mais UBS na faixa Alta
  • D3 inversamente correlacionado com renda → maior dispersão
  • Distribuição bimodal esperada (núcleo urbano vs periferia)
  • A escala de classes do JSON de referência (Baixa/Média/Alta) difere levemente dos limiares internos do pipeline — precisam ser sincronizados
