Metodologia

Fórmula do IVSaúde

IVSaúde = (score_D1 × 0,50) + (score_D2 × 0,30) + (score_D3 × 0,10) + (score_D4 × 0,08) + (score_D5 × 0,02)

Padronização

ModeloAplicado emGrau 0Grau I (0,25)Grau II (0,50)Grau III (0,75)Grau IV (1,00)
Modelo ID1, D2, D4, D5= 0< média≤ média×1,10≤ média×1,50> média×1,50
Modelo II (invertido)D3= 0 entidades< média≤ média+1DP≤ média+2DP> média+2DP

Próximos Passos para Dados Reais

  1. Obter shapefile oficial dos 141 territórios de UBS via GeoSampa (geosampa.prefpoa.com.br) ou SMS-POA
  2. Baixar setores censitários IBGE 2022 RS: python src/download_shapefiles.py --only ibge
  3. Baixar dados SIM / SINASC / SINAN: python src/download_shapefiles.py --only sim
  4. Obter cobertura ESF via e-Gestor APS: python src/download_shapefiles.py --only esf
  5. Executar pipeline com dados reais: python src/calcular_ivs.py
  6. O mapa interativo Folium (outputs/maps/ivs_poa_mapa_interativo.html) pode ser publicado em qualquer servidor web
